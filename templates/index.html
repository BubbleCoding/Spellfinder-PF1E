Pathfinder 1e Spell Search

Advanced Search

Type any words into the search box to search across spell names, descriptions, schools, components, and more. Words are matched as prefixes, so fire matches fireball, fire storm, etc.

Boolean operators

For more control, use AND, OR, NOT, or "quoted phrases" in your query. When these are present the search is passed directly to the full-text engine with no prefix expansion.

Field:value tokens

You can narrow your search with field:value tokens anywhere in the search box. Multiple tokens are combined with AND.

Data Storage

All your data — favorites and spellbooks — is stored in your browser's localStorage. Nothing is sent to any server; everything stays on your device.

  • Favorites (★) are a per-device list of spell IDs stored automatically when you star a spell.
  • Spellbooks are stored the same way. If you clear your browser's site data, all spellbooks will be lost — use Export Key to back them up.
  • Each browser profile has its own separate storage, so spellbooks created in Firefox won't appear in Chrome.

Spellbook Keys

A spellbook key is a JSON snapshot of a spellbook you can copy, save, or share.

  • Export Key — opens a dialog with the key text. Copy it and save it somewhere safe (a text file, a notes app, etc.).
  • Export List — opens a dialog with the spellbook's spell names, one per line. Useful for sharing a readable list or for printing.
  • Import Key — paste a previously exported key to restore or import a spellbook. If a spellbook with the same name already exists, you can rename it on import.
  • Import List — paste a plain list of spell names, one per line. Any names found in the database are added to the currently selected spellbook. Unrecognised names are reported so you can correct them.
  • Keys include the spellbook name, all spells in the book, and the prepared/unprepared status of each spell.
  • You can share a key with another player running Spellfinder — they import it on their own device and get a copy of your spellbook.
